# BACKEND REST_API Message Application

## Tentang Project

Ini adalah backend dari application message sederhana.
Pada aplikasi ini, menggunakan postgreSQL untuk databasenya
Aplikasi ini dibangun oleh **_Alpredo Pandiangan_** 

## Menjalankan Project
Untuk menjalankan project ini, bisa dengan :

### `nodemon` atau `node server.js` `atau npm start`

## Library yang digunakan

Adapun **dependencies** yang saya gunakan adalah :

`babel` untuk mengubah kode dari ES6 ke ES5 agar dapat berjalan di browser

`body-parser` untuk handle request data

`nodemon` untuk me-restart server setiap kali membuat perubahan pada file

`express` framework nodejs

`morgan` untuk monitoring request

`sequelize` perpustakaan untuk terhubung ke database postgreSQL

`sequelize CLI` yang memungkinkan kita untuk menjalankan migrasi basis data dengan mudah dari terminal dan bootstrap proyek baru.
